# DrawTabInventory

> ## ⚠ Deprecated — use [DrawTabDataExplorer](https://github.com/TheSevenPens/DrawTabDataExplorer) instead
>
> This project has been superseded by **[DrawTabDataExplorer](https://thesevenpens.github.io/DrawTabDataExplorer/)**, which offers a strict superset of features:
>
> - **[Tablet Inventory](https://thesevenpens.github.io/DrawTabDataExplorer/tablet-inventory)** — replaces this site's _Tablets_ tab
> - **[Pen Inventory](https://thesevenpens.github.io/DrawTabDataExplorer/pen-inventory)** — replaces this site's _Pens_ tab
>
> The Explorer's inventory pages add multi-field filters, a column picker, saved views, and direct links to the canonical pen/tablet detail pages.
>
> This repo is kept available for historical reference but will receive no further updates.

---

## What this project was

A single-page static web app that displayed a personal inventory of drawing tablets and stylus pens, loading JSON from the [DrawTabData](https://github.com/TheSevenPens/DrawTabData) submodule.

See [`docs/OVERVIEW.md`](docs/OVERVIEW.md) and [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md) for a full description of what was here.
